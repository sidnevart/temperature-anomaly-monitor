# temperature-anomaly-monitor
MVP Streamlit-дашборда для обнаружения аномалий температурных датчиков

## Rule-based detection

В проекте используется набор простых правил для объяснимого обнаружения температурных аномалий.

Правила обнаруживают:
1. Потерю сигнала — если датчик не передал значение.
2. Резкий скачок температуры — если изменение температуры между соседними точками превышает заданный порог.
3. Сильное отклонение от нормы — если z-score превышает допустимое значение.
4. Зависание датчика — если показания долго не изменяются.
5. Отклонение от группы датчиков — если один датчик сильно отличается от среднего значения по группе.
6. Медленный рост температуры — если средняя скорость изменения температуры за окно наблюдения положительная и температура выше rolling mean.

Rule-based detection нужен для понятных технологических тревог, которые можно объяснить оператору.  
Isolation Forest используется дополнительно для поиска нетипичного поведения, которое может быть сложно описать простыми правилами.
